# 产品需求文档

## 1. 核心业务流程（更新版）

1. **自由浏览**：网站一打开，默认直接进入首页（`index.php`）。游客（未登录用户）可以自由浏览商品列表，并能点击进入详情页（`detail.php`）查看商品参数和别人的历史评论。
2. **主动登录**：在首页或详情页的顶部导航栏，会有一个高亮的"登录"按钮。用户主动点击该按钮，才跳转到登录页（`login.php`）。
3. **权限控制（核心改变）**：
   - **未登录状态**：详情页底部的评论输入框和提交按钮**隐藏**，取而代之的是一条提示：*"您当前处于游客状态，请[点击登录]后发表评论。"*
   - **已登录状态**：导航栏的"登录"按钮变成"欢迎您，XXX | 退出"。详情页底部的评论表单**正常显示**，允许用户发表评论。

## 2. 页面功能与 UI 布局规范（更新版）

### 页面一：首页 (`index.php`)

- **顶部导航栏 (Navbar) 逻辑**：
  - 用 PHP 的 `if-else` 判断 Session。
  - `if` 未登录：右侧显示一个蓝色的"登录"按钮（Bootstrap `btn-outline-primary`），点击跳转到 `login.php`。
  - `else` 已登录：右侧显示 "欢迎您，[用户名]" 和一个红色的"退出"按钮。
- **商品列表**：无论登没登录，均正常循环渲染写死的商品卡片。

### 页面二：商品详情与评论页 (`detail.php`)

- **顶部导航栏**：与首页完全一致，保持全站状态同步。
- **评论列表区**：正常读取 MariaDB 并展示历史评论，游客也能看。
- **发表评论区（UI 条件渲染）**：
  - 用 PHP 判断 `if (!isset($_SESSION['user']))`：
    - 输出一个 Bootstrap 警告框（`alert alert-warning`），里面写："想要参与讨论？请先 [点击登录]"（文字加超链接指向 `login.php`）。
  - `else`（已登录）：
    - 输出 `<form>` 表单、文本域和"提交评论"按钮（`btn-primary`）。

### 页面三：登录页 (`login.php`)

- **前端 UI**：保持不变（居中卡片表单）。
- **后端逻辑**：登录成功并开启 Session 后，直接重定向回首页 `index.php`。

## 3. 后端部署环境

### 3.1 运行环境

| 项目 | 选型 |
|------|------|
| 宿主机 | VirtualBox 虚拟机 |
| 操作系统 | Rocky Linux 9.2 |
| Web 服务器 | Apache（httpd） |
| PHP 版本 | 8.0+（通过 Remi 仓库安装） |
| 数据库 | MariaDB 10.5+ |

### 3.2 安装依赖

```bash
# 1. 启用 EPEL 与 Remi 仓库
dnf install -y epel-release
dnf install -y https://rpms.remirepo.net/enterprise/remi-release-9.rpm

# 2. 安装 Apache + PHP 8.2 + MariaDB
dnf module enable -y php:remi-8.2
dnf install -y httpd php php-mysqlnd php-cli mariadb-server

# 3. 启动服务并设为开机自启
systemctl enable --now httpd mariadb
```

### 3.3 防火墙与 SELinux

```bash
# 开放 HTTP 端口
firewall-cmd --permanent --add-service=http
firewall-cmd --reload

# 调整 SELinux 上下文（如代码部署在 /var/www/html 下）
restorecon -Rv /var/www/html
```

### 3.4 数据库初始化

```bash
# MariaDB 安全初始化
mysql_secure_installation

# 导入建库脚本
mysql -u root -p < init.sql
```

### 3.5 代码部署路径

项目所有 PHP 文件放置于 Apache 默认文档根目录：

```
/var/www/html/
├── index.php
├── detail.php
├── login.php
├── logout.php
├── submit_comment.php

├── assets/

│       └── style.css
└── includes/
    ├── db.php
    ├── header.php
    ├── footer.php
    └── products.php
```
