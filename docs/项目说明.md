# 数码优选 — 项目说明

## 项目概述

一个基于原生 PHP + MariaDB 的数码产品展示与评论网站。用户可自由浏览商品列表和详情，登录后发表评论。

- **技术栈**：PHP（无框架）、MariaDB、Bootstrap 5、纯 CSS
- **字符集**：UTF-8（utf8mb4）
- **宿主机**：VirtualBox 虚拟机
- **操作系统**：Rocky Linux 9.2
- **Web 服务器**：Apache（httpd）
- **运行环境**：PHP 8.0+（Remi 仓库），MariaDB 10.5+

|------|------|---------|

## 核心业务流程

### 权限模型

```
         ┌──────────────┐
         │   访问网站     │
         └──────┬───────┘
                │
    ┌───────────▼───────────┐
    │   未登录（游客）        │
    │   导航栏："登录"按钮    │
    │   详情页：可看评论      │
    │   详情页：评论表单隐藏   │
    │   显示登录引导提示      │
    └───────────┬───────────┘
                │ 用户主动点击"登录"
    ┌───────────▼───────────┐
    │   登录成功              │
    │   → 重定向回首页        │
    └───────────┬───────────┘
                │
    ┌───────────▼───────────┐
    │   已登录                │
    │   导航栏："欢迎，XXX | 退出" │
    │   详情页：评论表单正常显示 │
    │   可提交评论            │
    └──────────────────────┘
```

### 页面流转

```
首页(index.php) ──点击卡片──▶ 详情页(detail.php)
     ▲                            │
     │◄── 返回首页 ◄──────────────┘
     │
登录页(login.php) ──登录成功──▶ 首页
退出(logout.php) ──销毁Session──▶ 首页
```

## 各页面功能说明

### 1. 首页 `index.php`

- 商品卡片网格布局（Bootstrap row/col，响应式：小屏 2 列 / 大屏 3 列）
- 每张卡片：商品图片 → 名称 → 描述 → 价格 → "查看详情"按钮
- 商品数据来自 `includes/products.php`（共 6 款商品）

### 2. 详情页 `detail.php`

- 左侧商品大图 + 右侧商品信息 + 规格参数表
- 底部评论区：
  - 展示历史评论列表（按时间倒序，从 MariaDB 读取）
  - **未登录**：显示引导提示"想要参与讨论？请先登录"
  - **已登录**：显示评论文本框 + 提交按钮
- 无效 ID 时显示错误提示

### 3. 登录页 `login.php`

- 居中卡片表单，含用户名 + 密码输入框
- 登录成功 → 写入 `$_SESSION['user']` → 重定向首页
- 登录失败 → 显示错误提示
- 已登录用户访问自动跳转首页
- 底部提示测试账号

### 4. 评论提交 `submit_comment.php`

- 仅接受 POST，需登录
- 参数校验（product_id > 0，content 非空）
- 写入 `comments` 表后重定向回详情页
- 未登录直接拦截跳转登录页

## 部署要点

### 环境准备（Rocky Linux 9.2）

```bash
# 1. 启用 EPEL 与 Remi 仓库
dnf install -y epel-release
dnf install -y https://rpms.remirepo.net/enterprise/remi-release-9.rpm

# 2. 安装 Apache + PHP 8.2 + MariaDB
dnf module enable -y php:remi-8.2
dnf install -y httpd php php-mysqlnd php-cli mariadb-server

# 3. 启动服务并设为开机自启
systemctl enable --now httpd mariadb

# 4. 开放防火墙
firewall-cmd --permanent --add-service=http
firewall-cmd --reload
```

### 部署步骤

1. 将项目文件复制到 Apache 文档根目录 `/var/www/html/`
2. 执行 `restorecon -Rv /var/www/html` 调整 SELinux 上下文
3. 运行 `mysql_secure_installation` 初始化 MariaDB
4. 执行 `mysql -u root -p < /var/www/html/init.sql` 导入数据库
5. 按需修改 `/var/www/html/includes/db.php` 中的数据库密码
6. 确保 PHP Session 目录可写
7. 浏览器访问虚拟机 IP 验证部署

## 启动命令

### 生产环境部署（Rocky Linux 9.2 + Apache）

```bash
# 1. 确保服务运行
systemctl status httpd mariadb

# 2. 部署代码到 Apache 根目录
cp -r * /var/www/html/
restorecon -Rv /var/www/html

# 3. 初始化数据库（仅首次）
mysql -u root -p < /var/www/html/init.sql

# 4. 按需修改数据库密码
vi /var/www/html/includes/db.php
```

浏览器访问虚拟机 IP（如 `http://192.168.56.101`）。

### 本地开发（PHP 内置服务器，仅限开发）

```bash
# 1. 启动 MariaDB
sudo systemctl start mariadb    # Linux
# net start MariaDB             # Windows

# 2. 初始化数据库（仅首次）
mysql -u root < init.sql

# 3. 启动 PHP 内置服务器
php -S localhost:8000
```

浏览器访问 `http://localhost:8000`。

> **注意**：
> - `includes/db.php` 默认连接 `127.0.0.1:3306`，用户 `root`，无密码。若数据库配置不同，先修改该文件再启动。
> - PHP 内置服务器仅用于开发，**不可用于生产环境**。

### 前端说明

本项目前端使用 **Bootstrap 5 + Bootstrap Icons CDN** 引入，无 `npm` / `webpack` / `vite` 构建流程。修改 `assets/css/style.css` 后刷新浏览器即可生效。
